# Z80ALL

Bill Shen's 25.175MHz VGA and PS/2 capable Z80 Machine.

## Memory map

Kernel
0000-7FFF	Kernel
8000-FFFF	Kernel (fixed)

User
0000-7DFF	Process
7E00-7FFF	Udata stash

## Hardware:

Currently this build supports
- Onboard video
- PS/2 port
- CF interface
- Time tick

make diskimage
